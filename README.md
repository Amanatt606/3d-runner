*  Game Concept

Title: Endless Runner

This game is a simple 3D runner where the player moves forward automatically and must avoid obstacles on the road. The level is divided into several sections with increasing difficulty. The player must reach the finish area without falling or hitting obstacles.

 * Player Objective

The main objective is to survive the level and reach the finish line.

Win condition: Reach the finish area
Fail condition: Fall off the platform or hit obstacles
  Controls
A / D or Arrow Keys — move left / right
Player moves forward automatically
 * Level Design

The level is built using primitive objects (cubes and planes).

It includes:

Start area
3 different sections
Finish area
Sections:
Section 1: Straight path with basic obstacles
Section 2: Narrow path and increased difficulty
Section 3: Turns and danger elements
